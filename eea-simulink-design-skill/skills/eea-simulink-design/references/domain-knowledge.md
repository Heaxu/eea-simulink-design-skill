# 电气工程及其自动化专业知识速查

## 建模总原则

- 先定义系统边界：电源、功率级、控制器、负载、测量环节、保护逻辑和输出指标。
- 明确模型层级：理论传递函数适合控制设计，平均模型适合控制环验证，开关模型适合纹波和器件应力，Simscape 物理模型适合多域耦合，phasor/RMS 模型适合电力系统稳态和暂态。
- 所有参数必须有来源：题目给定、教材典型值、数据手册、按额定值计算、或假设值。
- 常用验证指标：稳态误差、超调量、调节时间、上升时间、纹波、THD、效率、功率因数、转矩脉动、SOC 轨迹、保护动作时间。

## 电路与电力电子

- 基础规律：KCL、KVL、欧姆定律、功率平衡、瞬时功率 `p(t)=u(t)i(t)`、三相有功功率 `P=sqrt(3) U_L I_L cos(phi)`。
- Buck 变换器连续导通近似：`Vout = D Vin`，电感纹波 `DeltaIL = (Vin - Vout) D Ts / L`，电容纹波近似 `DeltaVo = DeltaIL / (8 C fs)`。
- Boost 变换器连续导通近似：`Vout = Vin / (1 - D)`，注意占空比接近 1 时电流应力和控制裕度会恶化。
- 逆变器常用控制：SPWM、SVPWM、电压闭环、电流内环、dq 解耦、锁相环 PLL、L 或 LCL 滤波。
- 仿真选择：控制算法先用平均模型，纹波、开关频率、THD 或器件应力再用开关模型。

## 电机与拖动

- 直流电机常用方程：`u = Ra ia + La dia/dt + Ke omega`，`Te = Kt ia`，`J domega/dt = Te - TL - B omega`。
- 异步电机毕业设计常见路线：V/f 调速、矢量控制、转差频率控制、直接转矩控制。
- PMSM 常见路线：Clarke/Park 变换，id/iq 电流内环，速度外环，SVPWM，`id=0` 或弱磁控制。
- 控制结构通常采用级联环：电流环最快，速度环较慢，位置环最慢。采样周期和带宽要有数量级区分。
- 结果图至少包含转速、电磁转矩、三相电流、dq 电流、负载扰动响应。

## 电力系统与新能源

- 使用标幺制时说明基准容量、基准电压、基准阻抗和换算关系。
- 故障分析关注三相短路、单相接地、两相短路、保护动作、暂态电流和电压跌落。
- 光伏模型关注 I-V/P-V 曲线、辐照度、温度、MPPT 算法、DC/DC 接口和并网逆变器。
- 常用 MPPT：扰动观察法、增量电导法。验证时改变辐照度和温度，观察最大功率点跟踪速度和振荡。
- 电池系统关注 SOC、OCV、内阻、CC-CV 充电、过流过压保护、充放电效率。

## 自动化与控制

- 控制器从简单到复杂选择：开环逻辑、PID、前馈加反馈、状态空间、MPC、模糊控制、滑模控制。
- PID 毕设要给出整定依据：经验法、临界比例法、根轨迹/频域设计、或基于仿真的参数扫描。
- Stateflow 适合离散逻辑：启停、故障、保护、模式切换、PLC 类顺序控制。
- 传感器与执行器要考虑限幅、采样、噪声、滤波、延迟和饱和。

## 常见论文论证句式

- "为兼顾仿真效率与控制规律分析，本文首先建立平均模型用于控制参数整定，再采用开关模型验证输出纹波与动态响应。"
- "由于题目未给出器件数据手册，本文参考同功率等级典型参数，并在仿真中进行参数扰动验证。"
- "仿真结果表明，系统在负载突变后能够恢复到设定值附近，说明所设计控制策略具有一定抗扰能力。"
